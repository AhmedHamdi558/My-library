# مكتبتي النصية

مكتبة شخصية لعرض الكتب والروايات والقصص — مبنية بـ HTML خالص وتعمل على GitHub Pages مباشرة بدون أي سيرفر.

---

## هيكل المشروع

```
/
├── index.html
├── README.md
└── Files/
    ├── engine.json          ← قاعدة بيانات الكتب
    ├── Images/              ← صور الأغلفة
    │   └── اسم-الكتاب.jpg
    └── Texts/               ← ملفات الكتب والأوصاف
        ├── اسم-الكتاب.txt
        └── اسم-الكتاب-desc.txt
```

---

## إضافة كتاب جديد

### 1. ارفع الملفات
- نص الكتاب → `Files/Texts/اسم.txt`
- وصف قصير (اختياري) → `Files/Texts/اسم-desc.txt`
- صورة الغلاف (اختياري) → `Files/Images/اسم.jpg`

### 2. أضف إدخالاً في `Files/engine.json`

```json
[
  {
    "title": "اسم الكتاب",
    "image": "Files/Images/اسم-الكتاب.jpg",
    "descFile": "Files/Texts/اسم-الكتاب-desc.txt",
    "file": "Files/Texts/اسم-الكتاب.txt",
    "color": "#3a1020"
  }
]
```

> **مهم:** المسارات كاملة كما تراها — لا تختصرها.  
> `image` و `descFile` اختياريان، يمكن حذفهما.

## تنسيق ملفات النص `.txt`

### الصفحات
افصل بين الصفحات بسطر يحتوي `*` فقط:

```
نص الصفحة الأولى...

*

نص الصفحة الثانية...
```

### الألوان والتنسيق

| الصيغة | النتيجة |
|--------|---------|
| `gr/نص\gr` | نص أخضر |
| `r/نص\r` | نص أحمر |
| `b/نص\b` | نص أزرق |
| `y/نص\y` | نص أصفر |
| `B/نص\B` | **نص عريض** |
| `I/نص\I` | *نص مائل* |

### مثال

```
كان الليل gr/هادئاً\gr بشكل غريب، والريح r/باردة\r تقرع النوافذ.

B/قال:\B I/"لن أعود."\I

*

الصفحة الثانية تبدأ هنا...
```

## التنزيل من القارئ

داخل القارئ يوجد ثلاثة أزرار:

| الزر | الصيغة | الوصف |
|------|--------|-------|
| PDF | `.pdf` | كتاب منسق بغلاف وألوان، يفتح نافذة الطباعة |
| MD | `.md` | Markdown مع الألوان كـ HTML مضمّن |
| TXT | `.txt` | الملف الأصلي كما هو |
